      <p align="center"><font size="+2"><b>MIME Media Types</b></font></p>




        
      
      
      
      <blockquote> 
		
          
        
        
        
        <p>[<a href="http://www.ietf.org/rfc/rfc2046.txt?number=2046">RFC2046</a>] specifies that Media Types (formerly known as MIME types) and Media Subtypes will be assigned and listed by the IANA.</p>




          
        
        
        
        <p>Procedures for registering Media Types can be found in 
          <a href="http://tools.ietf.org/html/draft-ietf-appsawg-media-type-regs-14">[RFC-ietf-appsawg-media-type-regs-14]</a>,
          <a href="http://www.ietf.org/rfc/rfc4289.txt?number=4289">[RFC4289]</a>, and
          <a href="http://www.rfc-editor.org/rfc/rfc6657.txt">[RFC6657]</a>. 
          Additional procedures for registering media types for transfer via Real-time Transport Protocol (RTP) can be found in <a href="http://www.rfc-editor.org/rfc/rfc4855.txt">[RFC4855]</a>.</p>





          
        
        
        
        <p>The following is the list of Directories of Content Types and Subtypes:</p>




		
          
        
        
        
        <p><a href="http://www.iana.org/assignments/media-types/application/">application</a></p>
